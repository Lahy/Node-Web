## Demo

*node 版本：Node.js 0.6.11*

### 启动项

1. Web
``` javascript
  $ node web.js // 127.0.0.1:3000
```
2. index
``` javascript
  $ node index.js // localhost:8888
```

### nvm

``` bash
  nvm list　　//查看目前已经安装的版本
  nvm list available //显示可下载版本的部分列表
  nvm install 10.15.0 //安装指定的版本的nodejs
  nvm use 10.15.0 //使用指定版本的nodejs
```
